# Content Formulas — Caption Frameworks by Format

## Instagram Feed Post (Static / Carousel)

### Formula: HOOK → VALUE → CTA
```
[Hook line — 1 sentence, max 125 chars before "more"]

[Body — 2–4 sentences delivering the value]
[Include 1 specific proof point]

[CTA — direct, outcome-led]

[Line break]
[Hashtags — 5–10, mix of niche + medium-volume]
```

### Hook formulas:
- Question: "Did you know that [surprising fact about product/service]?"
- Statistic: "[X]% of tennis players will experience elbow pain. Here's why."
- Contrarian: "Stop doing [common thing]. Do this instead."
- Outcome: "103 tests. One morning. Results the same day."

---

## Instagram Reel Caption

### Formula: SHORT HOOK → CONTEXT → CTA → HASHTAGS
```
[1–2 line hook that matches the video hook]

[1–3 sentences of context — expand on what the video covers]

[CTA — link in bio or direct action]

[5–15 hashtags]
```

Note: First 2 lines critical (shown before "more"). Match them to the video hook exactly.

---

## Instagram Carousel (swipe post)

### Slide structure:
- Slide 1: Cover — hook headline + visual (no text overload)
- Slides 2–5: Content — one point per slide, max 1–2 sentences + visual
- Last slide: CTA slide — clear action + handle/website

### Caption formula:
```
[Swipe prompt: "Swipe to see the [X] [things/reasons/steps] →"]

[1–2 sentence content tease]

[CTA + hashtags]
```

## Instagram Story

### Single story:
- Background fill (brand color or scene image)
- 1 line of text max (make it readable at small size)
- Interactive element: Poll / Question sticker / Link sticker / Countdown
- No more than 3 elements on screen

### Story sequence (3–5 slides):
- Slide 1: Hook/question
- Slides 2–3: Content/answer
- Slide 4–5: CTA + product/service

## Caption do's and don'ts

### DO:
- Lead with the strongest line (hook)
- Use line breaks to create white space (easier to read)
- Include one specific number or proof point per caption
- Match caption tone to the visual
- End with a clear CTA (not "what do you think?" — that's weak)

### DON'T:
- Start with "We" or the brand name
- Use generic hashtags only (#love, #instagood)
- Write walls of text — use short paragraphs
- Include multiple CTAs (one per post)
- Use emojis as filler — use only when they add meaning (RTP and Kmedtest are low-emoji brands)
