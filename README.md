# Total Trading — Claude Marketing Skills

Reusable Claude Code skills for the Total Trading multi-brand system (RTP Tennis, Kmedtest).

## Skills
- `content-pipeline` — full content production orchestrator
- `viral-video-creator` — short-form video scripts & storyboards
- `creative-production-brief` — TTS scripts + image prompts per scene
- `canva-edit-brief` — editor instructions for Canva video assembly
- `kie-tts` — text-to-speech via Kie.AI API
- `asset-logger` — logs completed assets back to Notion
- `brand-promo-strategy` — seasonal marketing strategy docs
- `meta-ads-strategist` — Meta campaign structure & creative briefs
- `notion-marketing-campaigns` — Notion content calendar setup

## Setup
Clone this repo, then copy skills to your Claude Code folder:
```bash
cp -r skills/* ~/.claude/skills/
```
Brand context files (brand.md, notion-ids.md, rules.md) are kept locally — never committed.
